Welcome to Trumark DB


Available Commands

[name:] Glob search for names of CARDs.

name:<card_name> Returns card_name,cost

e.g. name:Sol Ring

[text:] Searches for keyword in CARD text.

text:<card_text> Returns card_text, card_name

e.g. text:island walk

[mana:] Search for CARD with specific mana cost.

mana:<mana_type> Returns mana_type, card_name

e.g. mana:{U}{U}

[type:] Search for CARD with specified type

type:<card_type> Return card_type, card_name

e.g. type:creature

[sub:] Search for CARD with specified subtype.

sub:<subtype> Return subtype, card_name

e.g. sub:angel

[super:] Search for CARD with specified supertype.

super:<supertype> Return supertype, card_name

e.g. super:legendary

[color:] Search for CARD with one or more colors.

color:<color> Return color, card_name

e.g. color:r , color:rg

[set:] Search for a particular SET.

set:<set> Return set_id, set_name, card_name

e.g. set:5ED

[in:] Used to Search a CARD in SET.

in:<card_name> Return card_name, set_id

e.g.in:sol ring

[all] Used to get all continence of tables.

<table>:all Return contence_of_table

e.g. set:all

[rarity:] Search for CARDs of a specific rarity.

rarity:<rarity> Return rarity, set_id, card_name

e.g. rarity:rare

[fmt] Search for MTG format

fmt:<format> Return format, card_name

e.g. fmt:commander

Operators <,>,<=,>=,==

Used in numerical comparisons. For 'cost:' and 'set:'.

cost:<operator> <int> Return card_cost, card_name

e.g. cost:<10

set:<operator> <int> Return date, set_name

e.g. set:>2012

Chaining Commands

type:creature,sub:angel Returns card_type, card_name, subtype

type:creature,cost:>=2 Return card_type, card_name, cost, mana_type

in:sol ring,rarity:rare Return card_name, set_id
