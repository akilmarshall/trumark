Welcome to Trumark DB


Available Commands

[name:] Glob search for names of CARDs.

e.g. name:Sol Ring

[text:] Searches for keyword in CARD text.

e.g. text:island walk

[mana:] Search for CARD with specific mana cost.

e.g. mana:{U}{U}

[type:] Search for CARD with specified type

e.g. type:creature

[sub:] Search for CARD with specified subtype.

e.g. sub:angel

[super:] Search for CARD with  specified supertype.

e.g. super:legendary

[color:] Search for CARD with one or more colors.

e.g. color:r ,  color:rg

[set:] Search for a particular SET.

e.g. set:5ED

[in:] Used to Search a CARD in SET.

e.g.in:sol ring

[all] Used to get all continence of tables.

e.g. set:all

Operators  <,>,<=,>=,==

Used in numerical comparisons.

i.e.cost:<10
